A web-based decision-support tool for accommodation accessibility in Edinburgh

Data source

  • Accommodation data from OpenStreetMap via Overpass API (Feb 2026, ODbL).
  • Basemap tiles: Mapbox.

Project description

This interactive map supports trip planning by allowing users to filter accommodation by type and identify options within a fixed 500 m radius around selected destinations in Edinburgh.
